2. Database Schema Upgrade

Add status, demand_level, and last_updated to properties and trends for richer analytics:

alter table properties add column status text default 'available';
alter table market_trends add column demand_level text;
alter table market_trends add column last_updated timestamptz default now();

3. Manual Data Import Flow

Add a JSON upload tool → previews before inserting into Supabase

Optional “fetch data from Gemini Deep Search” text field (manual copy-paste allowed, no scraping)

That keeps you compliant (manual), but efficient.

4. Emotional Copywriting Framework

Define tone rules (this helps if you regenerate UI text later):

Arabic → emotional, poetic, high-trust (“نحوّل قرارات العقار لقرارات ثروة.”)

English → confident, luxury, investor-minded (“Turn property decisions into wealth strategy.”)

Avoid generic real estate phrases — make every word feel personal & visionary.
